# Implementation Scope in this ZIP

This codebase bundles the P0-01 ~ P0-06 foundation changes.

## Included

- Multi-organization case participation model (`case_organizations`, `organization_relations`)
- Organization signup governance (`organization_signup_requests` upgrades)
- Staff/client invitation upgrades (`invitations` evolution)
- Permission templates and membership permission overrides foundation
- Client billing, collection compensation, inter-org settlement schemas
- Common Case Shell UI scaffold and case module catalog
- Updated forms/actions/queries for the above foundations

## Not yet fully implemented

- P0-07 workflow tracks full UI
- Inbox workflow implementation
- Messages/Requests complete UX
- Full Client Portal UX
- Full Collections Workspace dashboards
- Export engine (xlsx/docx/pdf/hwpx)

These are intentionally deferred so the domain model can stabilize first.
