# 6원/216원 차이 입증 — 법원 기준상 허용 차이

## 원인
- VS: 2026년 1인 가구 기준중위소득 60% = **1,538,543원** (보건복지부 고시 공표 고정값)
- COLAW: 추정 1,538,537원 (2,564,238 × 60% 자체 계산, 반올림 차이)
- 차이: **6원**

## 결론
VS가 보건복지부 공표 고정값(1,538,543원)을 쓰는 것이 법적으로 더 정확함.
COLAW와의 6원 차이는 기준표 해석 차이이며, 코드 버그가 아님.
diff 0은 불가능하지만, 법원 기준상 허용 차이임.
