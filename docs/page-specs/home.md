# Page Spec: Home (`/`)

## 목적
- 비인증 사용자에게 서비스 가치와 시작 동선을 제공한다.
- 인증 사용자에게 즉시 업무 진입 링크를 제공한다.

## 노출 컴포넌트
- `BrandBanner`
- 시작하기 CTA
- 데모 버튼(비인증만)
- 플랫폼/업무 요약 카드(인증 시)

## 사용 데이터
- `getCurrentAuth()`
- `getDashboardSnapshot()`

## 클릭 이벤트
- 시작하기: 인증 여부에 따라 기본 앱 경로 또는 `/start`
- 데모 보기: `/demo`
- 카드 이동: 알림(`/notifications`), 사건(`/cases`) 등

## 예외 상태
- 대시보드 스냅샷 조회 실패: 빈 스냅샷으로 렌더 + 콘솔 오류 기록

## 권한 조건
- 전체 공개 페이지
- 인증 여부에 따라 컴포넌트 분기

## 완료 기준
- 모든 주요 링크가 `ROUTES`/`NAVIGATION_MAP`을 사용한다.
- 인증/비인증 분기별 클릭 동작이 문서와 동일하다.
